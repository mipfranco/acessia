# ♿ AcessIA

AcessIA é um chatbot inteligente com foco em acessibilidade, desenvolvido para auxiliar pessoas de forma simples, intuitiva e inclusiva.

## 🚀 Funcionalidades

* 💬 Chat com inteligência artificial
* 🎤 Reconhecimento de voz
* 🔊 Leitura de respostas por voz
* 🌙 Tema claro e escuro
* 🔠 Controle de tamanho da fonte
* 📱 Responsividade para celular e tablet
* ♿ Interface acessível

## 🛠 Tecnologias utilizadas

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express

### APIs

* OpenAI API

## 📱 Responsividade

O projeto foi desenvolvido para funcionar em:

* Desktop
* Celulares
* Tablets

## 🎯 Objetivo

Criar uma experiência mais acessível e amigável para usuários que precisam de suporte visual, auditivo ou interação simplificada.
